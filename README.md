# Scrap Raider
A RPG extraction fighter game built in the Godot game engine. With a focus on making it a mobile friendly game.

## Development stages
1. (current) building a 2D top down version of the game with the games to learn how to impliment the magority of the systems used in the final game as well as creating a frame work for easy play testing.
2. Rebuild the game but in a low poly 3D style, with just the core gameplay loop. This will include a base storyline, upgrade and weapon system, different dungeon enviroments/challanges and a homebase/hideout that can be upgraded with resources.
3. Create a PvP multiplayer areana with arcade style feature where players can fight each other for glory and loot.
4. Add guilds to the game along with related features such as: multi-player raids/dungeons, guild areas that need the whole guild to help upgrade and guild wars plus objectives.
